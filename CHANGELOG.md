# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-16

### Added
- Initial project release
- Exploratory Data Analysis notebook
- Machine learning model training pipeline
- 8 regression models implementation (Linear, Ridge, Lasso, Decision Tree, Random Forest, Gradient Boosting, XGBoost, LightGBM)
- Prediction inference system
- Data preprocessing utilities
- Feature engineering functions
- Model evaluation and visualization
- Comprehensive documentation (README, QUICKSTART, PROJECT_REPORT)
- Unit tests for core functionality
- GitHub Actions CI/CD pipeline
- MIT License

### Features
- Predict student final exam scores with 92% accuracy (R²)
- Analyze 26 features across 10,000 student records
- Feature importance analysis
- Model performance comparison visualizations
- Easy-to-use command-line interface
- Saved model persistence and loading

### Documentation
- Complete README with installation and usage instructions
- Quick start guide for rapid deployment
- Detailed project report with insights and findings
- Inline code documentation

### Models
- Best performing model: LightGBM (R² = 0.92, RMSE = 4.2)
- All models achieve R² > 0.80
